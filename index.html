Say hello to CSSstudies, the first CSS study group in Berlin.

CSSstudies will be a weekly study group for all things HTML & CSS. Following a curriculum, each week we will cover a certain aspect while building a website. This way, all the possibilities shall be explored in detail while providing instant practice.

Who is this for?

The group aims at people who want to learn about HTML and CSS from scratch and then dive into all the details. It is for everyone who wants learn within a group as well as in a coached and guided setup where there is time and space for each and every question.

Did you always want to learn building websites? Or dabbled around with it a bit, but never really found a good starting point where to continue? Or have a feeling of being overwhelmed? Or feel demotivated learning all by yourself? CSSstudies is for you then.

You already know about HTML and CSS? Maybe there are colleagues, friends, family, … for whom the group is suitable. Please spread the word, tell them about it and encourage them to join.

CSSstudies shall offer a welcoming and safe environment, aiming to be attentative towards explicit as well as the more subtle toxic behaviours that can put off other people. Details for a Code of Conduct are not yet in place, but will be announced soon.

The curriculum

There will be a curriculum. At first, we will work through Shay Howe’s Tutorial, but also spice it up with other tutorials where suitable – such as the CSSclasses materials or the learnlayout.com site or the flexbox froggy.

The idea for each week is to learn about a certain aspect, clear questions, and then instantly use the knowledge, play around with the options, break stuff, clear questions, rinse & repeat.

The general flow for each session is supposed to be like this:

  • Recap I: summarize the previous session
  • Work through the current chapter, address terminology, concepts, etc.
  • Code along, play around, break stuff
  • Do all this while addressing questions
  • Recap II: Round up what has been addressed during the session
  • Preview for the next session

The topics

We will cover all kinds of topics related to HTML and CSS:

  • Basic terminology, box model, …
  • Tools & useful helper sites, …
  • Layouts, positioning, …
  • HTML meta tags, semantic HTML, …
  • Typical website content, menus, forms, …
  • Transformations, animations, …
  • Cascade, specificity, parsing, resets, …
  • Typography, images, …
  • Variables, modules, Sass, …

(Not necessarily in this order. And no, this is by no means a complete list.)

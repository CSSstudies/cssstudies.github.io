Materials for Learning and Writing CSS

Editors

For writing HTML and CSS, you will need an editor. Popular choices among web designers and developers are Sublime or Atom. If you're on Windows, feel free to use Notepad (it's pre-installed). For Linux, Bluefish is a great editor.

Note: You can't use MS Word, LibreOffice, Pages, TextEdit, or similar word processing apps. Those apps are adding information to a file that will prevent the correct display of your HTML file.

Curious what this means? Open any word or RTF document in an editor and see for yourself :).

CSS-Tricks

CSS-Tricks is _the_ place to go when looking up CSS stuff. It has well written articles, often accompanied by code examples. Search the site for whatever you're working on and see for the results. These sections are especially recommended:

  • Almanach: Find all the CSS properties
  • Videos: Just like the articles, the videos cover all kinds of coding situations, easily explained.

MDN - Mozilla Developer Network

Note: The headline link sends you straight to the CSS reference

The MDN is an extensive pool of knowledge. You can read about everything HTML, CSS and JavaScript. Besides the references (= all the CSS properties, HTML tags, DOM, Web APIs etc.), there are also tutorials for learning web development and working with debug and developer tools.

Unfortunately, it can sometimes be hard to find the right information. So here are a few quick links:

  • HTML elements: Find all HTML tags
